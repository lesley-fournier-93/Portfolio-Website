# ✨ Developer Portfolio

A sleek and interactive **developer portfolio** built with pure **HTML, CSS, and JavaScript** — focused on modern UI, smooth animations, and a unique project showcase.

---

## 🧠 Concept

This project combines clean design with subtle interactivity to create a portfolio that feels modern and engaging without relying on frameworks.

The goal:  
→ **Stand out visually**  
→ **Keep it lightweight**  
→ **Show frontend skills clearly**

---

## ⚡ Features

- 🎯 Dynamic role/title animation (JavaScript-powered)
- 🖱️ Interactive buttons with smooth hover transitions
- 🌙 Modern dark UI with neon/glow accents
- 🧩 3D rotating project cards
- 🔄 Circular motion along the **Y-axis**
- 📱 Responsive layout
- 🔗 Social media integration

---

## 🎥 Highlight

The project section features a **3D rotating carousel**:

- Projects rotate in a circular formation  
- Movement happens along the **Y-axis**  
- Creates depth and a pseudo-3D effect  
- Fully handled with CSS + JavaScript
